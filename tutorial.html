SOBA Assistant Tutorial

  1. 🛠️ Run the Plugin Launch Eclipse using a plugin configuration with `soba.plugin` auto-started.
  2. 📁 Create an EMF Project Use Eclipse to create a new EMF modeling workspace.
  3. 📂 Load the Metamodel Import a `.ecore` file to define model structure.
  4. 🧱 Create a Model Instance Begin editing a model that conforms to your metamodel.
  5. 🔧 Enable SOBA Nature Right-click the project → "Add SOBA Nature" This activates chat + model view.
  6. 💬 Use the Assistant Type in the chat to ask GPT for model recommendations. View suggestions in the tree and apply them to your model!
