Welcome to SOBA Assistant

SOBA is an AI-powered plugin designed to enhance model-driven development within the Eclipse IDE. It integrates GPT-based Large Language Models (LLMs) to help you create, refine, and explore models such as UML, XMI, BPMN, and Ecore through natural language interaction.

Why use SOBA?

  • 💬 Chat-based assistant that understands and manipulates models using natural language
  • 🌳 Interactive tree view for browsing and applying AI-generated model suggestions
  • 🧠 Context-aware recommendations based on your active model, metamodel, configuration, modelling history, and currently selected element
  • 🛠️ Extensible support for different modelling languages via Eclipse extension points

Get started with the How to Start guide, or dive into practical examples in the Tutorial.
