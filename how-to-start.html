Getting Started with SOBA Assistant

SOBA Assistant is an Eclipse plugin that helps you model more effectively using the Eclipse Modelling Framework (EMF) combined with AI-powered support. Follow these steps to get started:

✅ Step 1: Create an EMF Project

  1. In Eclipse, go to File > New > Project...
  2. Select EMF Project under Eclipse Modeling Framework
  3. Name your project (e.g., HotelModel) and click Next
  4. Load your metamodel (.ecore file) and configure the generation model
  5. Click Finish

✅ Step 2: Enable SOBA Support

  1. Right-click your EMF project and select Properties
  2. Go to Project Natures
  3. Add SOBA Project Nature to enable the assistant

This will activate the SOBA Chat View and SOBA Results View in the IDE.

✅ Step 3: Use the Assistant

Once SOBA is active, your project gains the following:

  • A settings/ folder for configuration
  • A chats/ folder to store assistant conversations
  • The right panel shows the SOBA Chat View
  • The bottom panel displays the SOBA Results View

✨ Step 4: Ask for Recommendations

Type prompts like:

create 10 rooms with numbers from 102, 5 single price 100 and 5 double price 180

SOBA sends a structured query to the LLM, which considers:

  • ✔️ Your current model and metamodel
  • ✔️ Your serialization and type configuration
  • ✔️ The currently selected model element
  • ✔️ The chat history for contextual memory

✅ Final Notes

SOBA's architecture combines metamodel-driven validation with LLM-powered suggestions, enabling fast and precise modelling via natural language. It supports multiple modeling languages and encourages extensibility via plugin integration.
