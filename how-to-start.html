Getting Started with SOBA Assistant

SOBA Assistant is a modelling helper plugin integrated into the Eclipse IDE. Follow these steps to begin using it with your EMF (Eclipse Modeling Framework) projects.

üß© Step 1: Create an EMF Project

  1. In Eclipse, go to File > New > Project...
  2. Select EMF Project under Eclipse Modeling Framework
  3. Name your project (e.g., HotelModel) and click Next
  4. Load your metamodel (e.g., a .ecore file)
  5. Click Finish

‚öôÔ∏è Step 2: Enable SOBA Support

  1. Right-click the project > Properties
  2. Navigate to Project Natures
  3. Add SOBA Project Nature to the list
  4. This activates the SOBA interface (chat view + result view)

üß† Step 3: Use the Assistant

Now that the project has SOBA enabled:

  • A settings/ folder is created for assistant configuration
  • A chats/ folder stores assistant conversations
  • The right pane shows the SOBA Chat View
  • The bottom pane includes the SOBA Results View

üí¨ Step 4: Ask for Recommendations

Once the metamodel and model (e.g., XMI) are loaded:

  • SOBA reads the active model and metamodel
  • Start typing in the chat to receive intelligent suggestions
  • Responses are shown in the results view
  • You can drag & drop suggestions into your model

You can run multiple sessions across models and projects, and each session is stored for continuity.
