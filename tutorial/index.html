SOBA Assistant Tutorial

This tutorial will guide you through the essential steps to use SOBA Assistant effectively inside Eclipse. SOBA enhances modeling by combining Large Language Models with real-time analysis of your active project.

Understanding SOBA's Workflow

  • SOBA works with Eclipse EMF projects that use metamodels like .ecore.
  • It reads your current model (e.g. .xmi) and its corresponding metamodel.
  • The assistant creates prompts that include model context, configuration, and chat history.
  • GPT then generates intelligent suggestions or answers based on your query.

Using the Chat

After installing and activating SOBA Nature on your project, a chat view appears on the right side of the Eclipse interface. Here, you can type natural language instructions to create, modify, or inspect model elements.

Example commands:

  • Add 3 single rooms with price 80
  • Create a guest with name John Doe and assign to room 204
  • Show me options for reservation status

Reviewing Suggestions

The results view shows model recommendations structured in a visual tree. These suggestions can be:

  • Inspected and edited before applying
  • Dragged into the model editor
  • Filtered according to the selected element in the model

Working Across Projects

SOBA supports multiple sessions and keeps track of the project and model you are working on. Each session:

  • Stores chat history and context
  • Can be reopened and resumed
  • Supports seamless switching between models

You can explore SOBA’s full power by combining it with your custom metamodels and using the extension system to integrate additional modeling languages.
