What is the SOBA Plugin?

Power your Eclipse modeling environment with intelligent assistance, thanks to the SOBA Plugin, an AI-driven tool that integrates Large Language Models like GPT. Create, edit, and refine models such as UML, BPMN, XMI, and Ecore using natural language queries and chat-based interaction inside your IDE.

How it Works

  1. Context Analysis: Understands the active model, its metamodel, and serialization settings
  2. Prompt Building: Dynamically generates prompts for GPT based on the modeling context
  3. LLM Query: Sends prompt to GPT (OpenAI)
  4. Response Handling: Parses structured JSON recommendations
  5. User Interface: Chat + Tree view inside Eclipse for applying changes

Supported Modeling Types

  • XMI
  • BPMN
  • UML
  • Ecore

Support for more types is possible via an extension point system.
