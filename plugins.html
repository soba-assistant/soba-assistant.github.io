SOBA Plugins

SOBA is designed with a modular plugin architecture that allows support for different modelling languages and domain-specific behaviours. It integrates with Eclipse through extension points and enables easy customization of its assistant capabilities.

Core Plugin Responsibilities

Each SOBA plugin defines:

  • The set of element types it supports for GPT-based recommendations
  • The structure and rules of the metamodel
  • How to extract and serialize model data for GPT prompting
  • Response parsing and visualization logic for the Results View
